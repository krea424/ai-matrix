# Tech Stack: AI Matrix - Strategic Value Orchestrator

## 1. Frontend & Core Logic
*   **Framework:** [Next.js 14+ (App Router)](https://nextjs.org/) - For a high-performance, SEO-friendly (if needed), and scalable React foundation.
*   **Language:** [TypeScript](https://www.typescriptlang.org/) - Ensuring type safety for complex feasibility calculations and state management.
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) - Rapid, utility-first styling for the "Modern Dark" enterprise aesthetic.
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) - For smooth state transitions, "Reveal" animations, and interactive UI feedback.

## 2. Strategic Visualization
*   **Matrix Engine:** [D3.js](https://d3js.org/) - Bespoke implementation of the AI Strategic Matrix, allowing for custom coordinate mapping, interactive tooltips, and high-fidelity rendering.
*   **UI Components:** [Radix UI](https://www.radix-ui.com/) / [Shadcn UI](https://ui.shadcn.com/) - Unstyled, accessible primitives for sliders, dialogs, and navigation, styled with Tailwind to match our brand guidelines.

## 3. State & Data Management (Phase 1: Local-First)
*   **State Container:** [Zustand](https://github.com/pmndrs/zustand) - A lightweight, fast state management library perfect for handling the matrix's multidimensional variables.
*   **Persistence:** `localStorage` - Browser-based storage for persisting session data across refreshes.
*   **Portability:** **JSON Export/Import** - Custom utility to allow consultants to "Save" assessments as local files and "Load" them later or share them with colleagues.

## 4. Development & Tooling
*   **Icons:** [Lucide React](https://lucide.dev/) - Clean, consistent stroke icons for enterprise software.
*   **Linter/Formatter:** ESLint & Prettier - Standardized code quality.
*   **Version Control:** Git.
